# 內容簡介

《分散式系統常用技術及案例分析》全面介紹在設計分散式系統時所要考慮的技術專案，內容豐富、案例新穎，關聯理論與技術實作較為前瞻。《分散式系統常用技術及案例分析》不僅僅介紹了分散式系統的原理、基礎理論，同時還引入了大量市面上常用的最新分散式系統技術，不僅告訴讀者怎麼用，同時也分析了為什麼這麼用，並闡述了這些技術的優缺點。希望《分散式系統常用技術及案例分析》可以成為讀者案頭的工具書，供讀者隨手翻閱。

《分散式系統常用技術及案例分析》分為三大部分，即分散式系統基礎理論、分散式系統常用技術以及經典的分散式系統案例分析。第一部分主要介紹分散式系統基礎理論知識，複習一些在設計分散式系統時需要考慮的范式、知識點以及可能會面臨的問題，其中內含執行緒、通訊、一致性、容錯性、CAP理論、安全性和平行等關聯內容；同時講述分散式系統的常見架構系統，其中也內含最近比較火的RESTful風格架構、微服務、容器技術等。第二部分主要列出了在分散式系統套用中經常用到的一些主流技術，並介紹這些技術的作用和用法；這些技術涵蓋了分散式訊息服務、分散式計算、分散式儲存、分散式監控系統、分散式版本控制、RESTful、微服務、容器等領域的內容。第三部分選取了以淘寶網和Twitter為代表的中國外知名網際網路企業的大型分散式系統案例，分析其架構設計以及演變過程；這部分相當於是對第二部分零散的技術點做一個"串燒"，讓讀者可以結合技術的理論，看到實戰的效果。

《分散式系統常用技術及案例分析》主要面對的讀者是對分散式系統感興趣的電腦專業的學生、軟體專案師、系統架構師等。
